# Desafios-Curso-Python

## Desafio do ponto de cozimento da carne

Criar um programa que, dependendo da temperatura (em Celsius) do Steak, deve retornar o ponto de cozimento da carne em Português. O usuário deverá fornecer a temperatura.



## Desafio com Funções
Criar um programa que calcula a quantidade de tinta necessária para pintar uma parede. O usuário deverá fornecer as seguintes informações: rendimento, altura e largura.
 O programa deve mostrar na tela a mensagem "Você necessita de X latas de tinta".

 ## Calculadora de IMC
 Cálculo de IMC - Índice de Massa Corporal

 ### Entradas:
 - Altura em cm
 - Peso em Kg

 ### Saída
 - Menos que 18,5: Magreza
 - Entre 18,6 e 24,9: Normal
 - Entre 25,0 e 29,9: Sobrepeso
 - Entre 30,0 e 39,9: Obesidade
 - Maior que 40,0: Obesidade Grave
